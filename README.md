# DJ Ivan - Site Oficial

Site institucional moderno e imersivo do **DJ Ivan**, especializado em Agronejo (fusão de música eletrônica com sertanejo/agro).



## ✨ Funcionalidades

- **Design Dark Premium** com neon green e efeitos glassmorphism
- **Animações fluídas**: partículas no fundo, cursor customizado, vinil girando
- **Player de Música** completo com playlist (play/pause, progress bar, volume, next/prev)
- **Menu Hamburger** totalmente responsivo para mobile
- **Lightbox** para galeria de serviços
- **Formulário de Contato** com feedback
- **Efeitos Sonoros** nos botões (Web Audio API)
- **Totalmente Responsivo** (desktop + mobile otimizado)

## 🎨 Tecnologias Utilizadas

- **HTML5** + **CSS3**
- **JavaScript** (Vanilla)
- **Google Fonts** (Poppins + Inter)
- **Font Awesome** Icons
- Canvas para partículas de fundo
- Web Audio API

## 🚀 Como Rodar

1. Baixe ou clone o projeto
2. Coloque os arquivos de áudio na pasta `audio/` e as imagens na pasta `img/`
3. Abra o arquivo `index.html` no navegador

**Ou use Live Server (VS Code)** para melhor experiência.
